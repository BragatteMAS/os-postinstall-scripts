# macOS System Defaults
# Declarative macOS preferences management (inspired by nix-darwin system.defaults)
#
# Format: domain|key|type|value
# Types: bool, int, float, string
# Lines starting with # are comments, blank lines ignored
#
# Idempotent: only writes if current value differs from declared value
# Backup: current values are saved before changes for rollback
#
# Minimum macOS version tested: 13.0 (Ventura)
